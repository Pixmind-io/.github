## What is Pixmind?

Pixmind is an all-in-one AI creative platform that brings the world's best image and video models under one roof. Instead of juggling subscriptions and rewriting prompts for every model, you describe the outcome in plain language — Pixmind handles model routing, iteration, and production for you.

Built for creators, marketers, e-commerce teams, and developers.

## For Developers — API & Token Platform

Integrate Pixmind into your apps and pipelines with a REST API:

```bash
curl -X POST https://aihub-admin.aimix.pro/open-api/v1/image/generate \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $PIXMIND_API_KEY" \
  -d '{"prompt": "A cinematic portrait...", "model": "gpt-image-2"}'
```

## What is Pixmind?

Pixmind is an all-in-one AI creation platform that brings together the world's best image and video models under one roof. Generate stunning visuals from text prompts, transform photos into videos, apply artistic effects, and more — no design experience needed.

## API Access

Integrate Pixmind into your apps with a simple REST API:

```bash
curl -X POST https://aihub-admin.aimix.pro/open-api/v1/image/generate \
  -H "Content-Type: application/json" \
  -H "X-API-Key: $PIXMIND_API_KEY" \
  -d '{"prompt": "A cinematic portrait...", "model": "nano-banana-2"}'
```

- Pay-as-you-go, no subscriptions
- Supports text-to-image, image-to-image, and video generation

## Features

- **Text to Image** — Describe it, see it generated in seconds
- **Image to Image** — Transform existing images with AI
- **Text to Video** — Turn prompts into short video clips
- **Image to Video** — Animate any photo into a video
- **AI Effects** — Kiss, Hug, Inflate and more viral effects
- **Art Filters** — Sketch, anime, oil painting, watercolor, pixel art
